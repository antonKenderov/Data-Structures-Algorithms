# Arrays Introduction

- Arrays (масивите) организират данните подред с индекси.

| Operation | Complexity |
| --------- | ---------- |
| Lookup    | O(1)       |
| Push      | O(1)       |
| Insert    | O(n)       |
| Delete    | O(n)       |

## Static Array (`T[]`)

### ✅ Pros

- Точна дължина.
- Достъпване с индекс – **O(1)**.
- Използва малко по-малко памет.
- Полезно, когато знаем големината предварително.

### ❌ Cons

- Големината **не се сменя динамично**.
- Добавяне/премахване на елементи е **O(n)**.
- По-ограничено API.

### 📌 Кога се използва?

- Знаем броя на елементите предварително.
- Performance и memory са критични.

---

## Dynamic Array (`List<T>`)

### ✅ Pros

- Dynamic size.
- Богато API: `Add()`, `Remove()`, `Insert()`.
- Достъпване с индекс – **O(1)**.
- По-лесен за поддръжка.

### ❌ Cons

- Има малък memory overhead.

### 📌 Кога го използваме?

- Броят на елементите не е известен или се променя.
- Трябва често да добавяме или премахваме елементи.
- Искаме по-лесен и четим код.

---

# Имплементация на Array (`List`)

| Method              | Description                               |
| ------------------- | ----------------------------------------- |
| `get(index)`        | Връща стойността на даден индекс.         |
| `push(item)`        | Добавя елемент в края.                    |
| `pop()`             | Изтрива последния елемент.                |
| `delete(index)`     | Изтрива елемент по индекс.                |
| `shiftItems(index)` | Премества елементите след дадения индекс. |
